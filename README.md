## Usage

### Manual Installation

1. Copy and paste the JSON files (e.g. `catppuccin-mocha.json`) into `AppData\Local\imhex\themes`, then restart the program
2. Go to Help -> Settings -> Interface -> Color Theme and select your Catppuccin flavor
